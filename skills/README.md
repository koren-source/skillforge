# Community Skills

This directory is the starter library for SkillForge.

Each subfolder contains a portable `SKILL.md` file that an AI agent can load directly. The goal is to make it easy for the community to share high-signal operating knowledge, not generic summaries.

Current built-in examples:

- `meta-ads/` for Meta advertising frameworks, testing systems, scaling rules, and creative diagnostics
- `yc-fundraising/` for founder fundraising patterns, investor psychology, and YC-style application guidance

To contribute:

1. Add a new folder under `skills/`
2. Include a strong `SKILL.md` with practical guidance, not placeholders
3. Open a pull request with the source material or reasoning behind the skill
